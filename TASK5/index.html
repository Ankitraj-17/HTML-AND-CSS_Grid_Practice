Answers

1. Where does gap add space?

Gap adds space between grid items, not on the edges of the container.

2. How is gap different from margin?

Gap only adds space between items. Margin adds space around each item including edges.

3. Does gap affect container size?

No, gap works within the container. It doesn't change container size.

4. Can we use gap only for rows or columns?

Yes. Use row-gap for rows only and column-gap for columns only.

5. Why is gap preferred in Grid layouts?

Gap is cleaner and simpler. No need to worry about margin collapse or edge spacing.
