Answers

1. What does fr stand for?

fr stands for "fraction". It represents a fraction of the available space in the grid container.

2. How many columns are created?

3 columns are created because we used grid-template-columns: 1fr 1fr 1fr; which defines 3 equal columns.

3. What happens when screen size is reduced?

The columns shrink proportionally because fr units are flexible. They adjust based on available space.

4. Do items overflow or adjust?

Items adjust and shrink with the container. They don't overflow because fr is a flexible unit.

5. Why is fr better than px for layouts?

fr is better because it is responsive. px is fixed and doesn't change with screen size. fr automatically divides available space equally.
