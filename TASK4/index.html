Answers

1. What does auto do in grid rows?

auto adjusts the row height based on content. If content is small, row is small. If content is big, row grows.

2. Which row grows based on content?

The second row (Box 4, Box 5, Box 6) uses auto so it grows based on content.

3. Can rows have different heights?

Yes. First row is 100px fixed, second row is auto.

4. What happens if content is very large?

With auto, row expands. With fixed px, content overflows.

5. Where are grid rows useful in real layouts?

Useful for header (fixed), main content (auto), footer (fixed).
