Answers
1. What problem does auto-fit solve?

Auto-fit solves the problem of empty space. It fits as many columns as possible into the container and wraps items to new rows if there isn't enough space.

2. Why is minmax important?

Minmax ensures items have a minimum size (e.g., 200px) so they don't become too small, while allowing them to grow (1fr) to fill available space.

3. Do items shrink too much?

No, because minmax(200px, 1fr) prevents them from shrinking below 200px.

4. Is this layout mobile friendly?

Yes! It automatically adjusts from multiple columns on desktop to a single column on mobile without extra code.

5. Why are media queries not needed here?

Because CSS Grid logic (auto-fit + minmax) handles the responsiveness natively based on available width.
