Answers

1. Which layout breaks faster on small screens?

The fixed layout (px) breaks faster because the columns have fixed width and don't shrink.

2. Which layout adjusts automatically?

The flexible layout (fr) adjusts automatically because fr units divide available space.

3. Why is fixed layout risky?

Fixed layout is risky because on small screens it causes horizontal scrolling and content overflow.

4. Where can fixed columns be useful?

Fixed columns can be useful for sidebars, navigation menus, or when you need exact sizes like image thumbnails.

5. Which unit would you choose for a responsive website?

I would choose fr because it makes the website responsive and works on all screen sizes without breaking.
